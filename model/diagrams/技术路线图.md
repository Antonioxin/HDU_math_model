# 技术路线图（Mermaid 源）

> 渲染后导出为 PNG / PDF，保存至 `paper/figures/技术路线图.png`，由论文手在 `1_introduction.tex` 或 `3_model.tex` 引用。

```mermaid
flowchart TD
    subgraph DATA [数据层]
        D1[附件表 1]
        D2[附件表 2]
        D3[附件表 3]
    end

    subgraph Q1 [Q1 <任务名>]
        Q1A[预处理 / 特征]
        Q1B[模型求解]
        Q1C[结果检验]
    end

    subgraph Q2 [Q2 <任务名>]
        Q2A[输入构造]
        Q2B[模型求解]
        Q2C[对照与检验]
    end

    subgraph Q3 [Q3 <任务名>]
        Q3A[外层枚举]
        Q3B[内层重解 Q2]
        Q3C[决策输出]
    end

    RESULT([最终决策与论文产出])

    DATA --> Q1A --> Q1B --> Q1C
    Q1C -- "接口变量 1" --> Q2A
    Q2A --> Q2B --> Q2C
    Q1C -- "接口变量 2" --> Q3A
    Q2C -- "接口变量 3" --> Q3A
    Q3A --> Q3B --> Q3C
    Q3C --> RESULT

    classDef data fill:#eee,stroke:#888
    classDef q1   fill:#d6e8ff,stroke:#3a78c2
    classDef q2   fill:#ffe5cc,stroke:#cc7a00
    classDef q3   fill:#d8f0d8,stroke:#3a8a3a
    classDef out  fill:#fff5b5,stroke:#999

    class D1,D2,D3 data
    class Q1A,Q1B,Q1C q1
    class Q2A,Q2B,Q2C q2
    class Q3A,Q3B,Q3C q3
    class RESULT out
```
