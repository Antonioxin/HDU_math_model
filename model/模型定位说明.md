# 模型定位说明

> **维护者**：建模手。
>
> **目的**：声明各问的任务定位、求解方法、输入/输出，并给出跨问接口变量表，为论文手提供衔接描述的数学依据。
>
> **关联**：符号定义见 [`assumptions.md`](assumptions.md)；参数数值见 [`参数总表.md`](参数总表.md)；各问详细公式见 [`formulation/`](formulation/)。
>
> **任务-Q 映射**：本赛题共四个任务，按建模需求合并为三问：
> - **Q1 ↔ 任务一**：飞轮退化建模 + 当前健康状态初评估
> - **Q2 ↔ 任务二**：轴承振动特征提取 + 轴承退化建模
> - **Q3 ↔ 任务三 + 任务四**：跨域迁移学习 + 健康预警机制与管理报告

---

## 1. 总体链路

```
                        附件1（仿真全寿命 3500 d）        XJTU-SY（15 轴承振动）
                                  │                              │
                                  ▼                              ▼
                       ┌──────────────────┐          ┌────────────────────────┐
                       │  Q1 飞轮退化建模  │          │  Q2 轴承特征+退化建模  │
                       │  (物理指数+Wiener)│          │  (HI + Wiener)         │
                       └────────┬─────────┘          └────────────┬───────────┘
                                │                                  │
                  θ̂_T^local, (θ_H, θ_F) │                  θ̂_S, HI_b(t), 阶段标签│
                                │                                  │
                                └──────────────┬───────────────────┘
                                               ▼
                          附件2（在轨 1800 d）─▶│
                                               ▼
                                  ┌─────────────────────────────────┐
                                  │  Q3 跨域迁移 + 预警 + 健康报告  │
                                  │  (L1 时间对齐 → L2 形态对齐     │
                                  │   → L3 加权融合 → IG-RUL → L等级)│
                                  └─────────────────────────────────┘
                                               │
                                               ▼
                            最终输出：  附件2 飞轮 RUL ± 95% CI、阶段 Ŝ、预警 L̂、决策建议
```

**链路特性**：
- Q1 与 Q2 **并行独立**（不存在数据耦合，仅共享 [`assumptions.md`](assumptions.md) 中的符号与阶段划分准则）；
- Q3 是 Q1 与 Q2 的**汇合点**，按"L1→L2→L3"逐级强化迁移；
- 任务四的预警机制建立在 Q3 输出的 RUL 概率分布之上，无独立模型层。

## 2. Q1：飞轮退化建模与健康评估

### 2.1 任务定位

| 维度 | 描述 |
|------|------|
| 问题类型 | **回归 + 状态估计 + 预测**（参数估计 → 阶段划分 → RUL 外推） |
| 选用模型 | **模型 A：物理指数模型** $i(t)=i_0+\alpha(e^{\beta t}-1)+\varepsilon_t$<br>**模型 B：Wiener 漂移退化过程** $X(t)=X_0+\mu t+\sigma B(t)$，$X=\ln(i-i_0+\epsilon)$ |
| 输入 | 附件1：$\{(t_k, i_k, i_k^{\text{theo}}, T_k, \omega_k, T_{f,k})\}$，$k=0,\dots,175$（$t\in\{0,20,\dots,3500\}$）；附件2：$\{(t_k, i_k, i_k^{\text{theo}}, T_k, \omega_k)\}$，$k=0,\dots,1800$ |
| 输出 | $\hat{\boldsymbol\theta}_A,\hat{\boldsymbol\theta}_B$（飞轮退化模型参数）；附件2 飞轮 $\hat S(1800)\in\mathcal S$、$\widehat{\text{RUL}}(1800)$ 及 95% CI |

### 2.2 求解工具

| 工具 | 用途 | 备注 |
|------|------|------|
| Python + pandas | 数据加载/对齐 | 共用 `code/src/data/load_tables.py` |
| `scipy.optimize.curve_fit` | 物理指数模型 NLLS | Levenberg-Marquardt；初值见 [`参数总表.md`](参数总表.md) §2 |
| `numpy` 解析 MLE | Wiener 参数估计 | 一阶差分服从 $\mathcal N(\mu\Delta t,\sigma^2\Delta t)$，闭式解 |
| `scipy.stats.invgauss` | 首达时分布 | RUL 概率与 CI |
| `ruptures` (PELT) | 阶段突变点检测 | 与阈值法交叉验证 |

### 2.3 检验方式

- **数值口径检验**：参数估计后回代附件1，逐 20 天比对 $i_t$ 与 $\hat i_t$，RMSE / $R^2$ / AIC / BIC；
- **退化检验**：$\beta\to 0$ 时模型 A 应退化为线性 $i_0+\alpha\beta t$；
- **与基线对比**：与"常电流均值"基线对比 RMSE，要求降低 ≥ 50%；
- **A/B 一致性**：两模型对附件2 RUL 估计的偏差 ≤ 15%。

详见 [`边界检验设计.md`](边界检验设计.md) §2。

## 3. Q2：轴承振动特征提取与退化建模

### 3.1 任务定位

| 维度 | 描述 |
|------|------|
| 问题类型 | **特征工程 + 评价 + 退化建模** |
| 选用模型 | **特征评价**：Mon/Trd/Rob 综合得分 → Top-3 PCA-PC1 → HI<br>**退化建模**：Wiener（主） + 物理指数（对比） |
| 输入 | XJTU-SY 全 15 个轴承振动数据：双通道、25.6 kHz、每分钟一文件 |
| 输出 | 每轴承 HI 序列 $\{\text{HI}_b(t)\}$、阶段标签 $\hat S_b(t)$、可迁移特征集 $\mathcal F^\star$、轴承退化参数 $\hat{\boldsymbol\theta}_b$ |

### 3.2 求解工具

| 工具 | 用途 | 备注 |
|------|------|------|
| Python + scipy | 时/频/时频特征提取 | `scipy.signal.welch`、`scipy.fft` |
| `PyWavelets` | 小波包分解 | 4 层 db4 |
| `PyEMD` | 经验模态分解 / HHT | 取前 3 阶 IMF |
| `scipy.stats.spearmanr` | 趋势性指标 Trd | — |
| `sklearn.decomposition.PCA` | HI 主成分构建 | 取 PC1 |
| `ruptures` (PELT) | HI 突变点 → 阶段划分 | — |

### 3.3 检验方式

- **HI 单调性检验**：全寿命 Mon ≥ 0.85；
- **3σ 报警检验**：报警时刻应早于失效时刻 $\ge$ 寿命的 20%（早期敏感性）；
- **工况一致性**：三工况下退化模型形式（指数 / Wiener）相同，仅 $(\mu,\sigma^2)$ 数值不同。

详见 [`边界检验设计.md`](边界检验设计.md) §3。

## 4. Q3：跨域迁移学习 + 预警 + 健康报告（任务三 + 任务四）

### 4.1 任务定位

| 维度 | 描述 |
|------|------|
| 问题类型 | **域自适应回归 + 决策规则** |
| 选用模型 | **L1 时间尺度对齐**：$\tau=t/T^{\text{EOL}}$<br>**L2 形态对齐**：仿射映射 $\mu_T=a\mu_S+b$，$(a,b)$ 由附件1 早期数据匹配<br>**L3 加权融合**：$\hat{\boldsymbol\theta}_T=\lambda\hat{\boldsymbol\theta}_S+(1-\lambda)\hat{\boldsymbol\theta}_T^{\text{local}}$，$\lambda=\exp(-\text{MMD}^2/h)$<br>**RUL 推断**：Wiener 首达时 $\text{IG}$ 分布<br>**预警规则**：4 级 $L_0,\dots,L_3$ 由 $\hat S$ 与 $P(\text{RUL}<\Delta)$ 联合决定 |
| 输入 | Q1 输出 $\hat{\boldsymbol\theta}_T^{\text{local}}$；Q2 输出 $\hat{\boldsymbol\theta}_S$ 集合（按工况）、HI 序列；附件2 当前观测 $X(1800)$ |
| 输出 | 附件2 飞轮迁移后 $\widehat{\text{RUL}}^{\text{TL}}(1800)$、95% CI、当前预警等级 $\hat L(1800)$、健康管理报告 |

### 4.2 求解工具

| 工具 | 用途 | 备注 |
|------|------|------|
| `numpy` 自实现 MMD/CORAL | 域差异度量 | 高斯核 |
| `sklearn.manifold.TSNE` | 可迁移性可视化 | — |
| 自实现 TCA / 加权融合 | 迁移核心 | 参考 Pan et al. 2011 |
| `scipy.stats.invgauss` | RUL 概率 | — |

### 4.3 检验方式

- **退化检验**：$\lambda=0$ 时 Q3 输出 = Q1 输出（RUL 差异 < 1 天）；
- **改进对照**：Q3 vs Q1 的 PHM Score Function（非对称损失），要求 Q3 不劣于 Q1；
- **预警偏序检验**：$\hat S=H\Rightarrow\hat L\le L_1$，$\hat S=F\Rightarrow\hat L\ge L_2$，100% 满足。

详见 [`边界检验设计.md`](边界检验设计.md) §4。

## 6. 命名与约定

- 接口 CSV 一律按"问号_变量名.csv"命名（如 `Q1_theta_A.csv`），列顺序与本表"含义"列对齐；
- 接口 JSON 顶层键为变量名，值为对象；
- 参数估计的不确定性（Wiener 协方差矩阵、CI）以**额外列** `_lo/_hi` 形式存储；
- 时间列统一用 `t_day`（飞轮）或 `t_norm`（归一化寿命）；不混用单位。
