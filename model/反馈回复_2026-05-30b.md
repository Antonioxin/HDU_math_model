## 1. 根因：附件2 复用了附件1 的退化参数

### 代码现状（[run_q1.py:329-351](../code/src/q1/run_q1.py#L329-L351)）
```python
res_a = fit_model_a(t1, i1)                                   # ← 只拟合附件1
rul_a = predict_rul_model_a(t2[-1], res_a["alpha"], res_a["beta"])   # 用附件1 α,β 算附件2 RUL
res_b = fit_wiener_lambda(t2, i2, res_a["alpha"], res_a["beta"])     # 用附件1 α,β 给附件2 Wiener
# 注释原文："复用附件1 模型 A 的 (α, β) 作为 Λ(t) 参数"  ← 即偏离点
```

### 为什么错：附件2 是退化更快的独立轮子
我用附件2 数据独立拟合指数模型验证：

| 拟合来源 | α | β | R²(对附件2) | Λ(1800) vs Dc=0.895 | RUL_A |
|---|:---:|:---:|:---:|:---:|:---:|
| **复用附件1（现状）** | 0.598 | 3.58e-4 | **0.673** | 0.541 ≠ 0.895 ❌ | 1548 d |
| **附件2 独立拟合（正确）** | 0.333 | **7.19e-4** | **0.999** | 0.882 ≈ 0.895 ✅ | **483 d** |

- 附件2 退化速率 $\beta_T=7.19\times10^{-4}\approx 2\beta_1$ —— 附件2 比附件1 快近一倍；
- 复用附件1 的慢 β，使 $\Lambda(t_c)=0.54\not\approx D_c=0.90$，**破坏 §3.3.5 点估计一致性条件**，于是：
  - A/B 偏差 35%（模型 A 仍按附件1 曲线外推到 1548d，模型 B 从附件2 真实状态起算到 1000d）；
  - 残差 $r_k=D_k-\Lambda_1(t_k)$ 含**系统性漂移**（非噪声），把 $\hat\sigma_B^2$ 灌爆到 0.434（应为噪声量级 $\sim10^{-4}$）。

## 2. 修复：附件2 路径全部改用附件2 自身拟合

### 代码改动（`code/src/q1/run_q1.py::main`）
在第 339 行前**新增附件2 独立拟合**，并让 RUL_A、Wiener 全部用它：
```python
res_a = fit_model_a(t1, i1)            # 附件1 拟合（保留：供附件1 检验、阶段速率、Score）

# ★ 新增：附件2 独立拟合（spec Q1 §5.1）
res_a_t = fit_model_a(t2, i2)          # 预期 α_T≈0.333, β_T≈7.19e-4, R²≈0.999

    t2[-1], i2[-1], res_a_t["alpha"], res_a_t["beta"], res_b["sigma_B2"]
```
