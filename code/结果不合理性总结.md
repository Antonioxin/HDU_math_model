## ⚠️ 仍存问题 1：PHM Score = 22025（预期 O(1)~O(10)）

```
PHM Score (Q1) = 22025.5
PHM Score (Q3) = 22025.5
```

**根因**：评估时用**附件2 的 α,β** 预测**附件1** 中间时刻（t=1000,1500,2000,2500）的 RUL。两个飞轮退化速率差 2 倍（β_T≈2β₁），用快轮参数预测慢轮 → 系统性低估 → clip(130) 上限触发 → expm1(13)≈4.4×10⁵ 累加。

**解决方案**：评估时每个 cut-time 用附件1 自身截至该时刻的拟合参数，而非附件2 的。这样 ΔRUL≈0，Score 落到 O(1)。

---

## ⚠️ 仍存问题 2：σ_B² = 0.295（预期 ~1e-4）

```
附件2 σ_B² = 0.295
附件2 RMSE = 0.0081, R² = 0.9990
```

**根因**：Λ-时间 Wiener 方差公式 σ_B² = (1/K) Σ (Δr_k)² / ΔΛ_k 中，ΔΛ_k 全程在 0.0002~0.0009 之间。即使拟合残差极小（RMSE=0.008），除以 ΔΛ_k≈0.0005 也会将方差放大约 2000 倍。**这是公式的尺度特性，不是实现错误。**

**解决方案**：改用残差直接方差 σ_B² = Var(r_k) ≈ RMSE² ≈ 6.4×10⁻⁵，与指数模型 σ_ε² 量级一致。

## ⚠️ 仍存问题 3：敏感性扫描中 IG exp 溢出

```
RuntimeWarning: overflow encountered in exp
  return float(norm.cdf(z1) + np.exp(2.0 * lam / m) * norm.cdf(z2))
```

**根因**：当 i_F 扫描到极端值时，`lam/m = δD/σ_B²` 变得极大，`exp(2λ/m)` 溢出。不影响已写入的结果表。

**解决方案**：改用 `scipy.stats.invgauss` 替代自实现，或加 `clip(lam/m, 0, 500)`。
